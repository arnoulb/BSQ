## Programmation dynamique - Projet scolaire
Trouver la plus grande sous matrice carrée le plus rapidement possible

**Génération de map :**
```
./gen_board <largeur> <longueur> <densité>
```

**Fonctionnement :**

```
make
./bsq <path_to_map>
```
